Dragon's Den Gaming Café Website

Overview
This project is a multi-page website created for CMIS 410. The website represents a fictional business called **Dragon’s Den Gaming Café**, a space that combines tabletop gaming, arcade entertainment, and a café environment.

The goal of this project was to design a clean, modern, and consistent website while applying HTML and CSS concepts learned throughout the course.

Live Site
https://michaeloldham293.github.io/dragon-den-site-final/

Pages Included
- **Home (index.html)**  
  Overview of the business, featured services, and a contact form.

- **About (about.html)**  
  Background of the café, business concept, and embedded video.

- **Arcade (arcade.html)**  
  Information about the arcade area, featured image, and a styled pricing table.

- **Contact (contact.html)**  
  Dedicated contact form for customer inquiries.

Features
- Consistent navigation across all pages
- Responsive layout using CSS
- Embedded video (YouTube)
- Styled data table (arcade pricing)
- Functional contact form layout
- External links (Facebook, Instagram, Maps)
- Telephone and email links
- Image integration with alt attributes

Technologies Used
- HTML5
- CSS3
- GitHub Pages (for hosting)

Notes
All content in this project is fictional and created for educational purposes only.
